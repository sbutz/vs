# vs

Übungen zur Veranstaltung Verteilte Systeme


## Tipps:
Neues Projekt erstellen:
```
mvn archetype:generate -DgroupId=de.othr.vs -DartifactId=uebung1
```

Bauen:
```
mvn package
```

Ausführen:
```
java -cp target/Projekt-1.0.jar de.othr.vs.App
```
